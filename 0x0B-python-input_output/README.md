# 0x0B. Python - Input/Output
---
## Description

Project 11  in High Level Programming series is about:
* How to open a file, write text in a file, read full content of a file, move the cursor.
* Make sure file is closed after using it, using ```with``` statement
* What is JSON
* Serialization and deserialization
* How to convert Python data structure to JSON string and vice-a-versa.
---
